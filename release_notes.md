# Release Notes - ILCKL Membership App

## Version 1.0.0+1 (Initial Release)

Welcome to the initial release of the **ILCKL Membership** app! 

### ✨ New Features

* **Membership Management**: Core functionality to manage memberships and user profiles.
* **Authentication**: Secure login and authentication powered by Supabase.
* **Digital ID & QR Codes**: Dynamic QR code generation for digital membership verification.
* **Document Generation**: Export membership details and reports directly to PDF.
* **File Uploads**: Support for uploading and picking images and necessary files for your profile.

### 🛠️ Technical Details

* Built with Flutter.
* State management powered by `flutter_bloc`.
* Navigation handled seamlessly with `go_router`.
* Local secure data storage integrated.

---

*Note: Since this is the initial release of the application, all features are new. Please report any bugs or feedback to the development team!*
